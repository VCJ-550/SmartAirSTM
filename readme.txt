




控制舵机运转的关键点在于：如何产生特定占空比的PWM信号。

数字舵机的驱动方式与模拟一样，都是采用PWM信号驱动，但是数字舵机不需要提供连续的PWM，只需要提供一次就可以保持在对应角度。

STM32的系统时钟默认为72M，时钟周期为1/72，使用定时器的PWM输出功能，可以产生PWM信号，通过配置定时器计数频率和计数值，实现产生特定频率的方波信号。

数字舵机的PWM控制信号通常是周期T=0.02S，f=50Hz的PWM信号，旋转的角度由正脉宽决定，角度越大正脉宽就要越大。
